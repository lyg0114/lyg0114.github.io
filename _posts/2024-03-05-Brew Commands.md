### 패키지 설치
```bash
brew install [패키지명] 
```

### 서비스 관리
```bash
brew services start [서비스 이름]      # 서비스 시작
brew services stop [서비스 이름]       # 서비스 종료
brew services restart [서비스 이름]    # 서비스 재시작
brew services list                  # 서비스 리스트 출력 
```

### 설치된 패키지 목록 확인
```bash
brew list
```

### HomeBrew 업데이트
```bash
brew update
```

### 패키지 검색
```bash
brew search [패키지명] 
```
